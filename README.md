# 📌 Overview
This project contains basic Python programs to demonstrate:
- Conditional statements (if-else)
- Dictionary operations
- File handling (write & read)

---

# 📂 Programs Included

## 1. Grade Checker
### 🧠 Problem:
Take a score as input and print grade based on marks.

### 📊 Conditions:
- 90+ → A  
- 80–89 → B  
- 70–79 → C  
- 60–69 → D  
- Below 60 → F  

### 💻 Concept Used:
- if, elif, else

---

## 2. Student Grades System
### 🧠 Problem:
Create a dictionary to store student names and grades.

### ⚙️ Features:
- Add student and grade
- Update existing grade
- Display all students

### 💻 Concept Used:
- Dictionary
- Loop (while)
- Conditional statements

---

## 3. Write to a File
### 🧠 Problem:
Create a text file and write content into it.

### ⚙️ Functions Used:
- open() in write mode ("w")
- write()
- close()

---

## 4. Read from a File
### 🧠 Problem:
Read content from a text file and display it.

### ⚙️ Functions Used:
- open() in read mode ("r")
- read()
- print()

---

# 🚀 How to Run

Run each file using:

```bash
python3 grade_checker.py
python3 student_grades.py
